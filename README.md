# CRUD de Pessoa - Python

Projeto universitário de um sistema CRUD (Create, Read, Update, Delete) de pessoa utilizando Python e vetores para armazenamento de dados.

## Estrutura do Projeto

```
projeto-crud-pessoa/
├── src/               # Código fonte
│   ├── pessoa.py     # Classe Pessoa
│   ├── crud.py       # Operações CRUD
│   └── utils.py      # Funções auxiliares
├── tests/            # Testes
├── main.py           # Programa principal
└── README.md         # Este arquivo
```

## Como executar

```bash
python main.py
```
